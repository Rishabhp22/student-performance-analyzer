# Student Performance Analyzer

A Python-based Student Performance Analyzer that helps manage student records, perform academic analysis, and generate report cards using NumPy, CSV, and JSON.

## Features

* Create and manage classes
* Add student records
* Import student data from CSV files
* View all students
* Student-wise average calculation
* Subject-wise average calculation
* Topper detection
* Lowest performer detection
* Pass/Fail evaluation
* Grade generation
* Report card generation
* Search student by roll number
* Class-wise data storage

## Technologies Used

* Python
* NumPy
* CSV
* JSON
* File Handling
